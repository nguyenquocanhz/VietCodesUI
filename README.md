# VietCodeUI for Community (Windows Forms C#)

Chào mừng đến với **VietCodeUI for Community**, một dự án mã nguồn mở dành riêng cho việc xây dựng và chia sẻ các thành phần UI (User Interface) đẹp mắt và miễn phí cho **Windows Forms C#**.

## 🎯 Giới thiệu

🚀 **VietCodeUI for Community** được tạo ra với mục tiêu:

✅ **Tập trung hoàn toàn vào Windows Forms C#**: Chúng tôi hiểu rằng Windows Forms vẫn là một nền tảng quan trọng trong phát triển ứng dụng desktop tại Việt Nam.
✅ **Cộng đồng đóng góp**: Đây là nơi để các nhà phát triển C# cùng nhau chia sẻ kiến thức và tạo ra những thành phần UI chất lượng.
✅ **Miễn phí và mã nguồn mở**: Mọi người đều có thể sử dụng, đóng góp và học hỏi.
✅ **Thành phần UI tối ưu**: Chúng tôi cam kết mang đến những thành phần UI được thiết kế tỉ mỉ, đáp ứng nhu cầu của các nhà phát triển Windows Forms.

## ✨ Tính năng

🔹 Bộ sưu tập các thành phần UI dành riêng cho Windows Forms C#: Nút, biểu mẫu, bảng, v.v.
🔹 Thiết kế phù hợp với giao diện người dùng truyền thống của Windows.
🔹 Khả năng tùy biến cao, dễ dàng tích hợp vào các ứng dụng hiện có.
🔹 Hiệu suất tối ưu cho ứng dụng desktop.
🔹 Tài liệu và ví dụ minh họa chi tiết.

## 📥 Cài đặt

1️⃣ Đảm bảo bạn đã cài đặt **Visual Studio** với hỗ trợ phát triển **Windows Forms C#**.
2️⃣ Tải xuống mã nguồn của **VietCodeUI for Community** từ GitHub.
3️⃣ Thêm các thành phần UI vào **Toolbox** của Visual Studio.
4️⃣ Sử dụng các thành phần UI trong dự án Windows Forms C# của bạn.

## 🤝 Hướng dẫn đóng góp (Fork & Pull Request)

Chúng tôi rất hoan nghênh và trân trọng mọi sự đóng góp từ cộng đồng! Để đóng góp các thành phần UI mới hoặc sửa lỗi, bạn hãy làm theo các bước chuẩn dưới đây:

### 1️⃣ Fork dự án
Truy cập vào trang kho lưu trữ (Repository) gốc [VietCodesUI](https://github.com/nguyenquocanhz/VietCodesUI) và nhấn nút **Fork** ở góc trên bên phải để sao chép dự án về tài khoản GitHub cá nhân của bạn.

### 2️⃣ Clone mã nguồn về máy
Mở terminal/git bash và sao chép kho lưu trữ đã fork của bạn về máy (thay `username` bằng tài khoản GitHub của bạn):
```bash
git clone https://github.com/username/VietCodesUI.git
cd VietCodesUI
```

### 3️⃣ Cấu hình upstream (Để đồng bộ mã nguồn)
Để có thể cập nhật các thay đổi mới nhất từ kho lưu trữ gốc, hãy thêm liên kết upstream:
```bash
git remote add upstream https://github.com/nguyenquocanhz/VietCodesUI.git
```

### 4️⃣ Tạo nhánh mới (Branch)
Trước khi chỉnh sửa code, hãy luôn cập nhật nhánh `master` mới nhất và tạo một nhánh làm việc mới:
```bash
# Đồng bộ với upstream
git checkout master
git pull upstream master

# Tạo nhánh mới cho tính năng của bạn (ví dụ: feature/ModernButton)
git checkout -b feature/ten-component-moi
```

### 5️⃣ Thực hiện thay đổi & Commit
Thêm các thành phần UI mới hoặc sửa đổi code của bạn, sau đó lưu lại các thay đổi:
```bash
# Xem các file đã thay đổi
git status

# Thêm tất cả thay đổi vào hàng đợi
git add .

# Ghi nhận các thay đổi kèm thông điệp rõ ràng
git commit -m "Bổ sung component ModernButton và hiệu ứng hover"
```

### 6️⃣ Push lên kho lưu trữ đã Fork của bạn
Đẩy nhánh chứa các thay đổi của bạn lên GitHub:
```bash
git push origin feature/ten-component-moi
```

### 7️⃣ Tạo Pull Request (PR)
1. Truy cập vào kho lưu trữ đã Fork của bạn trên trình duyệt web GitHub.
2. Bạn sẽ thấy một thanh thông báo màu vàng gợi ý tạo PR: Nhấn vào nút **Compare & pull request**.
3. Điền tiêu đề và mô tả chi tiết các thay đổi/tính năng bạn vừa bổ sung (ví dụ: mô tả cách sử dụng các control mới, các thuộc tính tùy chỉnh).
4. Nhấn **Create pull request** để gửi yêu cầu đóng góp. Chúng tôi sẽ review code của bạn sớm nhất có thể!
